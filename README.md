# PyTorch ML and Deep Learning Modeling Lab

Consolidated modeling lab covering preprocessing, neural networks, CNNs, RNNs, LSTMs, autoencoders, transformers, and ViT experiments.

## Overview

This repository combines the machine learning and deep learning work into one complete modeling lab. The work moves from preprocessing and baseline modeling into neural networks, convolutional models, recurrent models, autoencoders, transformer text models, and Vision Transformer classification.

The combined structure shows the full modeling progression in one place, from data preparation to architecture comparison and evaluation.

## Problem

Modeling work becomes fragmented when preprocessing, training, and evaluation are separated across too many small projects. This lab keeps the learning path, model families, data preparation decisions, and evaluation work together.

## Scope

- Classical preprocessing and supervised learning
- Fully connected neural networks and CNN optimization
- VGG, ResNet-style, RNN, and LSTM experiments
- Autoencoder anomaly detection
- Transformer text summarization and encoder probing
- Vision Transformer image classification

## Approach

- Grouped related notebooks by model family
- Kept preprocessing and evaluation visible for each task
- Compared model behavior through metrics and training plots
- Separated heavy outputs and model weights from source code
